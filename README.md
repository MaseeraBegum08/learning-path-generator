#  AI Learning Path Generator

A Streamlit app that generates personalized learning roadmaps using AI.

## Features
- Custom roadmap based on skill level
- Weekly study plan
- PDF download
- AI Mentor Q&A
- Progress tracker

## Installation
pip install streamlit groq fpdf

## Run
streamlit run app.py

## Note
Set your GROQ API key before running:
set GROQ_API_KEY=your_api_key_here
